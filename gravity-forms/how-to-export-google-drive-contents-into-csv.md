# How to export Google Drive contents into CSV

In this article, we'll show you how to get a list of URLs from your Google Drive folders.

## Step 1: Create a Google Sheet

## Step 2: Install the add-on "Drive Explorer" or "Drive Mapper"

1. In Google Sheets, go to **Add-ons** → **Get add-ons**.

    ![get-add-ons](C:\Users\Glaiza\Projects\samples\gravity-forms\get-add-ons.png)

2. Search for "Drive Explorer".
3. Install the add-on, and then allow access to your Google Drive.



## Step 3: Generate the CSV data

1. Open **Extensions** → **Drive Explorer** → **Start export**.
2. Choose **Selected items**. 

    *Note: If you select "All files," Drive Explorer will fetch all files from your Drive.*

3. The sheet will be populated with the following columns:

    - Folder name
    - Folder Path
    - Filename
    - Direct Link
    - File Link
    - Viewers
    - Editors
    - Share Status

4. Download the file as Comma Separated Values (.csv).
