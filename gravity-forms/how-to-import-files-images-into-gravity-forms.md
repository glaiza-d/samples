# How to import files or images into Gravity Forms

In this article, we'll walk you through the steps to import images into Gravity Forms. Your CSV file must contain the image URLs for the images you want to import.
1. In Gravity Forms, go to **Import/Export**.

    ![gravity-forms-import](/images/gravity-forms-import-1.png)

2. After that, click **Import Entries**.

    ![gravity-forms-import-entries](/images/gravity-forms-import-entries.png)

3. Upload the CSV file.

    ![gravity-forms-upload-CSV](/images/gravity-forms-upload-CSV.png)

4. Choose an existing form or create a new one.

    ![gravity-forms-choose-form](/images/gravity-forms-choose-existing-or-new.png)


5. In this example, the "Image URL" field in the CSV file contains a list of URLs.

    ![gravity-forms-field-type](/images/gravity-forms-field-type-file-upload.png)

6. Choose **File Upload** from the "Field Type" drop-down menu.

    ![gravity-forms-file-upload](/images/gravity-forms-file-upload.png)

7. After that, click **Continue with Import**.

    ![gravity-forms-continue-import](/images/gravity-forms-continue-import.png)

8. Configure the import options if necessary. Otherwise, click **Create Form and Continue**
    **With Import**.

    ![gravity-forms-create-form](/images/gravity-forms-create-form-continue-import.png)

9. Next, you will get a confirmation message that the import is finished.

    ![gravity-forms-view-imported-records](/images/gravity-forms-view-imported-records.png)
    
10. To see all the files that were imported, click **View Imported Records**.
